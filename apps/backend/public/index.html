📋 خطوات التجربة:

  1. امسح رمز QR أعلاه لربط WhatsApp Web
  2. أرسل رسالة إلى رقم WhatsApp المتصل بالنظام
  3. انتظر الرد التلقائي من الذكاء الاصطناعي
  4. جرب رسائل مختلفة مثل:
    • "مرحبا، كيف حالك؟"
    • "ما أطول نهر في العالم؟"
    • "أريد معلومات عن منتجاتكم"
    • "شكراً لك"

🔗 API Endpoints المتاحة:

  • GET /api/health - فحص حالة النظام
  • GET /api/bots - معلومات البوتات
  • POST /api/chat - محادثة مباشرة
  • GET /api/whatsapp-web/status - حالة WhatsApp
  • POST /api/whatsapp-web/send - إرسال رسالة
